# 3. Publicación de la Documentación con GitHub Pages

## 3.1 Configuración para el despliege de la página

1. Abrir el repositorio en GitHub y acceder a **Settings**.  
2. En el menú lateral, hacer clic en **Pages**.  
3. Seleccionar la rama **gh-pages** 
4. Guardar los cambios con el botón **Save**.

## 3.2 Comprobación de la página

En github esperas a que en **deployments** carge la página y una vez carga pinchamos y nos la abre.
